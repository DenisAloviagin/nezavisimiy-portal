# Независимый Портал — бот

Telegram-бот, который каждый день собирает новости о наркополитике и психоделиках со всего мира, пишет заметки на русском и присылает их тебе на одобрение.

## Как работает

1. Каждый день в 12:00 (Бразилия) бот парсит RSS-ленты + делает веб-поиск через Claude
2. Claude отбирает 3-7 лучших материалов и пишет заметки
3. Бот присылает тебе в личку каждую заметку с кнопками ✅ / ❌
4. Нажимаешь ✅ — пост уходит в канал. ❌ — пропускается.

## Деплой на Render

### 1. Создай репозиторий на GitHub
Залей три файла: `bot.py`, `requirements.txt`, `render.yaml`

### 2. Подключи на Render
- Зайди на render.com
- New → Blueprint
- Подключи репозиторий
- Render сам создаст два сервиса из render.yaml

### 3. Добавь переменные окружения
В каждом сервисе в разделе Environment добавь:

| Переменная | Значение |
|---|---|
| BOT_TOKEN | токен от BotFather |
| ADMIN_CHAT_ID | 631093482 |
| CHANNEL_ID | -1004289103735 |
| ANTHROPIC_API_KEY | твой ключ от Anthropic |

### 4. Добавь бота в канал
В Telegram: настройки канала → Администраторы → добавь @nezavisimiy_portal_bot с правом публиковать сообщения.

## Где взять ANTHROPIC_API_KEY

Зайди на console.anthropic.com → API Keys → Create Key
